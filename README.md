🛠️ Git & GitHub: Da Teoria à Prática (Téo Me Why)
Este repositório contém meus estudos, anotações e exercícios práticos do curso de Git e GitHub ministrado pelo Téo Calvo. O foco aqui não é apenas "salvar arquivos", mas entender o ciclo de vida do desenvolvimento de software e análise de dados de forma colaborativa.

📌 O que estou aprendendo:
Configuração de Ambiente: Configuração de user.name, user.email e chaves SSH.

Ciclo de Vida do Arquivo: Untracked ➜ Staged ➜ Unmodified ➜ Modified.

Gestão de Commits: Criação de mensagens semânticas e versionamento por intenção.

Ramificações (Branches): Criação e navegação entre branches para isolamento de funcionalidades.

Colaboração: fetch, pull, push e a resolução estratégica de conflitos.

🚀 Diferenciais do Curso
O aprendizado com o Téo Me Why me proporcionou uma visão de "Dados em Produção":

Terminal First: Todo o gerenciamento é feito via linha de comando (Bash/MINGW64), garantindo maior controle sobre o repositório.

Boas Práticas: Uso de .gitignore para evitar o envio de arquivos sensíveis ou desnecessários (essencial para conformidade com LGPD).

Documentação: A importância de um README bem estruturado para o portfólio de um Analista de Dados.
